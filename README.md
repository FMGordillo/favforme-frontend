## Getting started

You should have installed:

- [asdf](https://asdf-vm.com/#/)
- (Optional) [Docker](https://docs.docker.com/get-docker/)
- (Optional) [direnv](https://direnv.net/)

1. `asdf install`
2. `npm install`
3. Run your server, and then `npm run dev`
